# chat_de
### 프로젝트 설명  
- 인턴 과정에서 경북대학교 총 동창회 어플리케이션의 채팅 기능을 구현했습니다.
- 총 동창회 어플리케이션의 여러 기능 중 , 채팅 기능만 동작할 수 있도록 한 레포지토리입니다.
### 주요 기능 소개  
- 사용자 구분 : 관리자/ 일반사용자, 회원의 기수별 채팅 구분  
- 개인 및 그룹 채팅  
- 메시지 및 파일전송 : 사진, 동영상, 파일  
- 채팅방 생성 / 강제 삭제  
- 푸시알림 : 채팅방 알림 켜기/끄기  
- 안읽은 메세지 개수 표시  
- 보낸 메세지 삭제  
- 강제 퇴장

### 성과  
- 경북대학교 총동창회 어플리케이션에 개발한 채팅 기능이 포함되어 **구글 플레이스토어**에 출시되었습니다.
